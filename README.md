# LR(0) Parser Implementation

## Overview
This project is an implementation of an LR(0) Parser developed for the Compiler Construction course. The parser demonstrates fundamental compiler design concepts including grammar processing, LR(0) item generation, DFA construction, parsing table generation, and shift-reduce parsing.

The project helps in understanding how compilers perform syntax analysis and validate input strings using LR parsing techniques.

---

## Features
- Context-Free Grammar Processing
- LR(0) Item Set Construction
- Closure and GOTO Operations
- DFA State Generation
- ACTION and GOTO Parsing Tables
- Shift-Reduce Parsing
- String Acceptance/Rejection Checking
- Syntax Analysis Demonstration

## Sample Workflow
1. Input Grammar
2. Generate LR(0) Items
3. Construct DFA States
4. Build Parsing Table
5. Parse Input String
6. Accept or Reject String
